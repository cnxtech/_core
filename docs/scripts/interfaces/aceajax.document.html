Interface Document

Contains the text of the document. Document can be attached to several EditSessions. At its core, Documents are just an array of strings, with each row in the document matching up to the array index.

indexToPosition

  • indexToPosition(index: number, startRow: number): Position
    • Defined in types/typings/ace/ace.d.ts:425

    Converts an index position in a document to a {row, column} object. Index refers to the "absolute position" of a character in the document. For example:

    var x = 0; // 10 characters, plus one for newline
var y = -1;

    Here, y is an index 15: 11 characters for the first row, and 5 characters until y in the second.

    Parameters

    • index: number

      An index to convert

    • startRow: number

    Returns Position
